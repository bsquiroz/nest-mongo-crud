# Ejecutar en desarrollo

1. Clonar el proyecto
2. Ejecutar el

```bash
pnpm install
```

3. Levantar la base de datos

```bash
docker-compose up -d
```

4. Contruir la data

```bash
http://localhost:3000/api/seed
```

## Stack usado

- Nest
- MongoDB
